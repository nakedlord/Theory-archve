# R Theory Cross-links

Статус: active cross-link map
Дата: 2026-04-25

## Назначение

Этот файл фиксирует связи R-теории с другими модулями архива.

Цель — сохранить связи, но не смешивать границы теорий.

Главное правило:

```text
cross-link is not merge
```

Связь между теориями не означает, что одна теория становится частью другой.

---

## 1. R Theory ↔ Culture as Externalized Instructions

### Статус

```text
active cross-link
```

### Главная связь

R-теория описывает уровни обучения, выбора и перепрошивки правил.

Culture as Instructions описывает, как инструкции выходят за пределы отдельного мозга и живут в агентах, текстах, ритуалах, артефактах и институтах.

### Формула

```text
R1 -> единичные инструкции
R2 -> цепочки инструкций
R3 -> анализ и перепрошивка инструкций
```

### Связанные файлы

- `../03_CULTURE_AS_INSTRUCTIONS/CANON.md`
- `../03_CULTURE_AS_INSTRUCTIONS/VERSIONS/instruction_network_full.md`
- `../90_CROSS_LINKS/R_THEORY__CULTURE_AS_INSTRUCTIONS.md`

### Граница

Культурная теория не заменяет R-теорию.

Она описывает внешний слой передачи инструкций.

## 3. R Theory ↔ PNA

### Статус

```text
active hypothesis cross-link
```

### Главная связь

PNA может быть возможным когнитивным механизмом временной сборки такого окна.

### Формула

```text
PNA may support ΔT, but PNA is not equal to ΔT
```

### Связанные файлы

- `../05_PNA/CANON.md`
- `../05_PNA/HYPOTHESES/pna_as_delta_t_basis.md`

## 5. R Theory ↔ SynTax

### Статус

```text
infrastructure cross-link
```

### Главная связь

SynTax является архитектурой управления мышлением, миграцией, каноном, патчами и проверкой.

R-теория является объектной теорией внутри архива.

### Формула

```text
SynTax manages R Theory; SynTax is not R Theory
```

### Связанные файлы

- `../02_SYNTAX/CANON.md`
- `../02_SYNTAX/VERSIONS/runtime_pipeline_full.md`
- `../02_SYNTAX/VERSIONS/validation_layer_full.md`
- `../00_META/MIGRATION_MODE.md`

### Граница

SynTax не должен подменять содержание R-теории.

Он задаёт правила работы с ней.

## 6. R Theory ↔ Boundary Saturation

### Статус

```text
separate module / no active theoretical merge
```

### Главная связь

Boundary Saturation находится в отдельном физически-спекулятивном модуле.

Do not use Boundary Saturation to support R Theory.
Do not use R Theory to support Boundary Saturation.

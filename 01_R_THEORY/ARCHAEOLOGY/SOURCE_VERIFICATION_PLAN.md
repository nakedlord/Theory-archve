# Archaeology Source Verification Plan — R Theory

Статус: active source verification plan
Дата: 2026-04-25

## Назначение

Этот файл задаёт план проверки археологического блока R-теории по источникам.

Цель — отделить:

- археологический факт;
- интерпретацию;
- R-гипотезу;
- speculative extension.

---

## 1. Правило проверки

Для каждого археологического узла нужно собрать:

```text
source -> factual claim -> interpretation -> R-reading -> limitation
```

Нельзя начинать с R-интерпретации и подбирать под неё факты.

---

## 2. Категории уверенности

### supported

Есть хорошие источники и слабые альтернативные объяснения.

### plausible

Идея совместима с источниками, но требует осторожности.

### contested

Есть значимые альтернативные объяснения или спорный материал.

### speculative

Идея интересная, но пока недостаточно опирается на источники.

### needs source

Нужен источник до использования в сильном выводе.

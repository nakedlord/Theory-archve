## Назначение

Этот файл индексирует археологическую ветку R-теории.

Он показывает:

- какие археологические узлы уже перенесены;
- какой статус у каждого файла;
- какой R-уровень предполагается;
- что файл усиливает;
- что требует источниковой проверки;
- какие риски интерпретации есть.

---

## 1. Общий принцип археологического чтения

Артефакт сам по себе не доказывает R-уровень.

R-интерпретация строится по связке:

```text
материал -> действие -> повторяемость -> передача -> вариативность -> перенос -> стабилизация
```

Чем больше признаков повторяемости, передачи, последовательности, вариативности и переноса, тем сильнее R-интерпретация.

## Назначение

Эта папка хранит подробную археологическую ветку R-теории.

Цель — перенести уже обсуждённые наработки по истории артефактов, а не развивать новые версии теории.

## Принцип чтения

Артефакт сам по себе не доказывает R-уровень.

Смотрим на связку:

```text
материал -> действие -> повторяемость -> передача -> вариативность -> перенос -> стабилизация
```

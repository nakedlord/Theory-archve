# Archaeology Falsification Map — R Theory Migration Draft

Статус: migration draft / needs source verification

## Назначение

Этот файл переносит правило: археологический маркер не должен автоматически подтверждать R-уровень.

Нужно фиксировать, что может ослабить или отменить R-интерпретацию.

---

## 1. Главный принцип

```text
artifact evidence is not direct mind evidence
```

Артефакт — это след действия, материала и контекста.

R-уровень — интерпретация по совокупности признаков.

---

## 2. Общая схема проверки

Для каждого маркера спрашиваем:

1. Есть ли повторяемость?
2. Есть ли передача?
3. Есть ли последовательность?
4. Есть ли вариативность?
5. Есть ли перенос принципа?
6. Есть ли социальная стабилизация?
7. Есть ли более простое объяснение?

## 10. Sapiens / Neanderthals networks

### R-чтение

network stabilization hypothesis.

### Что ослабляет

- нет значимого различия в сетях передачи;
- различие лучше объясняется климатом или демографией;
- новые данные показывают устойчивые дальние сети у neanderthals;
- различия в инновациях оказываются следствием сохранности материала;
- индивидуальная когнитивная разница объясняет больше, чем сеть.

---

## 11. Brain size steps

### R-чтение

speculative link with R2/R3 capacity.

### Что ослабляет

- объём мозга плохо коррелирует с археологическими маркерами;
- изменения объясняются телом, климатом, питанием;
- организация мозга важнее объёма;
- нет совпадения тайминга;
- культурная сеть объясняет больше, чем индивидуальный объём.

---

## 12. Итоговое правило

R-интерпретация усиливается не одним маркером, а сходимостью:

```text
повторяемость + передача + последовательность + вариативность + перенос + стабилизация
```

R-интерпретация ослабляется, если более простое объяснение покрывает материал без обращения к более высокому уровню.

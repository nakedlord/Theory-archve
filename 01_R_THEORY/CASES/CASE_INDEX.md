# R Theory Case Index

Статус: active case index
Дата: 2026-04-25

## Назначение

Этот файл индексирует кейсы R-теории.

Кейс не является доказательством теории. Кейс показывает, как теория работает на материале, где она усиливается и где остаётся слабой.

## 1. `ochre_symbol_to_concept.md`

### Статус

```text
draft case / supports symbol_to_concept
```

### Проверяемый патч

- `PATCHES/2026-04-25_symbol_to_concept.md`

### Суть

Охра помогает проверить лестницу:

```text
материал -> функция -> значимость -> символ -> переносимое различение
```

### Что усиливает

- различение функции, значимости, символа и понятия;
- осторожное чтение археологических маркеров.

### Что не доказывает

- что всякое использование охры символично;
- что охра автоматически означает понятие;
- что есть развитая R3-система.

### Нужно проверить

- практические функции охры;
- контексты использования;
- повторяемость;
- связь с телом, местом, ритуалом.

## 2. `fire_symbol_to_concept.md`

### Статус

```text
draft case / supports symbol_to_concept
```

### Проверяемый патч

- `PATCHES/2026-04-25_symbol_to_concept.md`

### Суть

Огонь показывает путь:

```text
технология -> центр лагеря -> значимость -> символ -> понятия дом/жизнь/граница
```

### Что усиливает

- связь технологии и символизации;
- роль огня как инфраструктуры обучения;

- что любой огонь символичен;
- что огонь сам по себе доказывает R3;
- что лагерь автоматически означает понятийное мышление.

## 3. `burial_symbol_to_concept.md`

### Статус

```text
draft case / supports symbol_to_concept and shadows_of_R3
```

### Проверяемые патчи

- `PATCHES/2026-04-25_symbol_to_concept.md`
- `PATCHES/2026-04-25_shadows_of_R3.md`

### Суть

Погребение показывает переход:

```text
практическая проблема -> значимость -> ритуал -> символ -> возможное понятие смерти
```

### Что усиливает

- различение санитарной функции и символического слоя;
- смерть как узел значимости;
- понятие смерти как переносимое различение.

### Что не доказывает

- религию;
- развитую теорию загробного мира;
- R3 по одному захоронению.

## 4. `shadows_of_R3_burial_case.md`

### Статус

```text
draft case / supports shadows_of_R3 candidate
```

### Проверяемый патч

- `PATCHES/2026-04-25_shadows_of_R3.md`

### Суть

Погребение может быть тенью R3: стабилизацией поведения вокруг смерти, где прозрачная причинная модель недоступна.

### Что усиливает

- shadows_of_R3 как candidate;
- связь смерти, ритуала и неопределённости;
- осторожное чтение ритуала не как науки и не как бессмыслицы.

### Что не доказывает

- что все погребения являются тенями R3;
- что ритуал содержит причинную теорию;
- что есть развитый символизм.

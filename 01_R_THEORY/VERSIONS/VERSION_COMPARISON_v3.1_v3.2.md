## 2. Главное отличие

v3.1_full фиксирует широкий слой R-теории:

- R0/R1/R2/R3;
- ΔT;
- сон;
- воображение;
- рефлексия;
- осознанность;
- R3 modes;
- дисциплина;
- аффект;
- археология;
- культура как инструкции.

v3.2 добавляет и канонизирует точное различение:

```text

```

Главная формула v3.2:

```text

```

Эти формулировки пока не вносить в `CANON.md` без отдельного решения.

Нельзя механически переносить всё из v3.1_full в active canon.

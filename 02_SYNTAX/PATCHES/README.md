# SynTax Patches

Эта папка хранит ключевые патчи архитектуры SynTax.

Патчи нужны не для красоты, а для трассировки: какая архитектурная функция была добавлена, зачем, какой дефект закрывала и как изменила работу системы.

## Текущая серия

- v1.9 — Infrastructure Split
- v2.6 — Context Arbitration Step
- v2.8 — Error E11 Context Lock-In
- v3.0 — Progress Delta Check
- v3.3 — Canon Aging
- v3.5 — Exploration Budget
- v3.6 — Module Independence
- v3.7 — Error Recurrence Counter
- v4.3 — Drift Detection Protocol
- v4.6 — Complexity Governance Rule
- v4.9 — Operator Registry Protocol

## Правило

SynTax-патч считается перенесённым, если есть:

- короткое описание проблемы;
- добавленная конструкция;
- функция конструкции;
- риск или ограничение;
- связь с другими слоями.

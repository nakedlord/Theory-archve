# Concrete Meme Synthesis Examples — MEAT Migration Draft

Статус: migration draft / needs concrete source cases

## Назначение

Этот файл переносит типы конкретных синтезов мемов, которые должны использоваться для будущей проверки MEAT.

Файл не является исчерпывающим набором кейсов.

---

## 1. Герой + моральное правило

Синтез:

```text
образ героя + правило поведения -> пример для подражания
```

MEAT-вопрос:

```text
почему образ и правило начинают передаваться вместе?
```

---

## 2. Страдание + смысл

Синтез:

```text
страдание + интерпретация -> инструкция терпения / верности / жертвы
```

Такой комплекс может быть очень устойчивым, если сцеплен с ритуалом или институтом.

---

## 3. Лозунг + идентичность

Синтез:

```text
короткая фраза + групповая принадлежность -> политический или культурный пароль
```

Связанный файл:

- `political_slogan_metric_read.md`

---

## 4. Ритуал + объяснение

Синтез:

```text
повторяемое действие + объяснение -> устойчивый ритуальный комплекс
```

Если объяснение исчезает, действие может жить дальше, но комплекс мутирует.

---

## 5. Символ + институт

Синтез:

```text
символ + роль + норма -> институциональный знак
```

Институт удерживает символ и контролирует его допустимые значения.

---

## 6. Образ врага + правило поведения

Синтез:

```text
образ врага + инструкция защиты / атаки / исключения
```

Такой комплекс может иметь высокую силу сцепления, если связан со страхом и групповой идентичностью.

---

## 7. Миф + архетип

Синтез:

```text
конкретный миф + повторяемая структура роли -> культурный архетип
```

Связанный файл:

- `cultural_archetype_metric_read.md`

---

## 8. Бренд + стиль жизни

Синтез:

```text
товарный знак + поведенческий сценарий -> идентичность потребителя
```

Статус: возможный будущий кейс, не активный теоретический узел.

---

## 9. Интернет-мем + социальная норма

Синтез:

```text
шутка + повторение + узнаваемая ситуация -> новая норма интерпретации
```

Статус: возможный будущий кейс.

---

## 10. Что проверять в каждом кейсе

- какие элементы сцеплены;
- что является ядром;
- что может мутировать;
- какова стоимость передачи;
- что удерживает эмоциональный заряд;
- есть ли институт;
- есть ли ритуал;
- есть ли пародии или локальные версии.

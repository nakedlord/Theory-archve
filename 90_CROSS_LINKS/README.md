# Cross-links

Эта папка хранит связи между теориями.

Связь не должна теряться внутри одного канона. Если одна теория использует механизм другой, это фиксируется здесь отдельным файлом.

## Зачем это нужно

- не дублировать одни и те же идеи в разных теориях;
- видеть, какие узлы являются центральными;
- отслеживать конфликты между теориями;
- отделять самостоятельную теорию от применения другой теории;
- не допускать скрытого смешения рамок.

## Правило

Если связь между теориями становится важной для вывода, создаётся отдельный файл:

```text
THEORY_A__THEORY_B.md
```

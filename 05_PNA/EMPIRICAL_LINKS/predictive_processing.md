# Empirical Link: predictive_processing

Статус: source-based bridge

## Цель

Связать PNA и R-теорию с исследованиями предсказания, ошибки и коррекции поведения.

## Различение уровней

```text
predictive processing -> ожидание и ошибка -> интерпретация через PNA -> возможная связь с R-уровнями
```

## Что даёт predictive processing

Predictive processing рассматривает восприятие и действие через ожидания, предсказания и ошибки предсказания.

Для R-теории это важно, потому что выбор и обучение тоже можно читать как работу с ожиданием результата и коррекцией после несовпадения.

## Связь с PNA

PNA может быть рабочим окном, в котором удерживаются:

- текущая ситуация;
- ожидание;
- возможное действие;
- ошибка;
- коррекция;
- новое правило.

## Связь с R-уровнями

R1: ожидание ближайшего результата действия.

R2: ожидание последовательности результатов.

R3: удержание ошибки и изменение правила, которое породило ошибку.

## Связь с ΔT

Если ΔT — окно выбора, то predictive processing помогает описать, что происходит внутри окна:

```text
ожидание -> действие или внутренний прогон -> ошибка -> коррекция -> стабилизация
```

## Чего это не доказывает

Этот мост не доказывает PNA напрямую.

Он не доказывает R-теорию.

Он только показывает, что в существующих когнитивных теориях есть близкая линия: мозг и поведение можно описывать через предсказание, ошибку и коррекцию.

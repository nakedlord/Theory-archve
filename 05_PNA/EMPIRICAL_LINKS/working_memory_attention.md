# Empirical Link: working_memory_attention

Статус: source-based bridge

## Цель

Связать PNA с рабочей памятью и вниманием без преждевременного переноса PNA в канон.

## Различение уровней

```text
исследования рабочей памяти и внимания -> интерпретация через PNA -> возможная связь с ΔT
```

## Что дают исследования рабочей памяти

Рабочая память описывает временную доступность информации для текущей задачи.

Это важно для PNA, потому что PNA предполагает временную сборку актуального паттерна: восприятие, внимание, память, ожидание и действие должны быть удержаны вместе.

## Что даёт внимание

Внимание ограничивает и выбирает то, что попадает в актуальное окно обработки.

Это важно для PNA, потому что без отбора внимания паттерн не может быть устойчивым рабочим окном.

## Осторожная гипотеза

PNA можно читать как временно собранный рабочий паттерн, который делает возможным:

- удержание текущей ситуации;
- удержание альтернатив;
- сравнение вариантов;
- стабилизацию выбора;
- переход от реакции к внутреннему прогону.

## Связь с ΔT

ΔT в R-теории — функциональное окно выбора.

Рабочая память и внимание могут быть частью механизма, который делает такое окно возможным.

Поэтому связь лучше формулировать так:

```text
working memory + attention -> temporary cognitive availability -> PNA -> possible basis for ΔT
```

## Чего это не доказывает

Этот мост не доказывает, что PNA уже является установленным нейрокогнитивным механизмом.

Он только показывает, какие существующие исследовательские линии могут поддерживать гипотезу.

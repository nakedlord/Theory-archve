# Empirical Link: temporal_integration_window

Статус: source-based bridge

## Цель

Проверить, можно ли связать PNA с исследованиями временной интеграции опыта.

## Различение уровней

```text
исследования временной интеграции -> осторожная интерпретация через PNA -> возможная связь с ΔT
```

## Что даёт temporal integration

Исследования временной интеграции показывают, что восприятие и сознательный опыт могут собираться не как мгновенная точка, а как короткое временное окно.

Для PNA это важно, потому что PNA предполагает временно собранный паттерн: элементы восприятия, памяти, внимания и ожидания удерживаются вместе достаточно долго, чтобы стать рабочей ситуацией.

## Окно 2–3 секунды

В литературе есть линия работ о субъективном настоящем и временных окнах порядка нескольких секунд.

Важно: это не нужно сразу превращать в жёсткий закон.

Для нашей модели это пока исследовательская опора, а не доказательство PNA.

## Связь с ΔT

ΔT в R-теории — окно выбора.

Temporal integration может быть нижним слоем для понимания того, как система удерживает ситуацию достаточно долго, чтобы сравнивать альтернативы.

Рабочая формула:

```text
temporal integration -> temporary present -> PNA -> possible support for ΔT
```

## Чего это не доказывает

Этот мост не доказывает, что все решения происходят в одном универсальном окне.

Он не доказывает, что PNA равен temporal integration.

Он только показывает, что у гипотезы PNA есть возможная связь с исследованиями субъективного настоящего и временной сборки опыта.

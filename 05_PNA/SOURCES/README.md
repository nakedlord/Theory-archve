# Sources — PNA

Статус: source map

## Зачем этот файл

PNA пока не должен быстро переходить в канон.

Сначала нужен слой источников, который отделяет:

```text
что известно из исследований -> как мы это интерпретируем -> как это связано с ΔT
```

## Рабочие группы источников

### 1. Working memory and attention

Нужно собрать источники о связи рабочей памяти, внимания и удержания актуальных представлений.

Полезные направления:

- working memory as temporary availability of information;
- attention as selection;
- attention as limited resource;
- focus of attention;
- cognitive control.

### 2. Temporal integration windows

Нужно собрать источники о временной интеграции восприятия и возможном окне 2–3 секунды.

Важно: не считать 2–3 секунды доказанным универсальным окном. Есть поддерживающие исследования и критика.

### 3. Predictive processing

Нужно собрать источники о мозге как системе предсказания и коррекции ошибки.

Это может быть связано с R-теорией через внутренний прогон, ожидание результата и коррекцию поведения.

## Правило чтения

Источник не доказывает PNA напрямую.

Источник может поддерживать один из мостов:

- внимание удерживает и выбирает актуальные представления;
- рабочая память делает информацию доступной для текущей задачи;
- временная интеграция собирает опыт в короткие окна;
- предсказательная обработка связывает ожидание, ошибку и коррекцию.

Только после этого можно осторожно связывать PNA с ΔT.

# Sleep, Imagination, Reflection — PNA / R Theory Migration Draft

Статус: migrated from memory / needs source verification

## Назначение

Этот файл переносит связку:

```text
сон -> воображение -> рефлексия -> осознанность
```

как разные режимы вынесенного из реальности обучения.

Файл связан с R-теорией и PNA, но не переводит PNA в канон.

---

## 1. Главная идея

R-уровни читаются как уровни обучения вне прямой ошибки в реальности.

Сон, воображение и рефлексия — разные способы уменьшить цену обучения.

Они позволяют системе перерабатывать опыт, строить будущие траектории и изменять правила поведения без немедленной платы реальным действием.

---

## 2. Сон

Сон — первичный офлайн-ускоритель обучения.

Он работает с прошлым опытом, памятью и эмоциональными следами.

Сон не обязательно адресен.

Он может перестраивать связи, не задавая явной цели.

В R-терминах сон ближе к неадресной переработке прошлого.

---

## 3. Воображение

Воображение — локальная симуляция будущих траекторий.

Оно использует память, но направляет её на возможное будущее.

Воображение позволяет проигрывать варианты до действия.

В R-терминах воображение связано с R2:

```text
ситуация -> возможная траектория -> ожидаемый результат
```

---

## 4. Рефлексия

Рефлексия — адресное обучение второго порядка.

Она анализирует не только действие, но и правило генерации действия.

В R-терминах рефлексия связана с R3:

```text
ошибка -> правило -> изменение правила
```

---

## 5. Осознанность

Осознанность — навигация по рабочей памяти.

Она позволяет выбирать, что удерживается в фокусе, что сравнивается и что становится объектом анализа.

Осознанность не равна R3, но может поддерживать R3-процессы.

---

## 6. Связь с PNA

PNA может быть рабочим окном, в котором удерживается:

- фрагмент сна;
- воображаемая траектория;
- ошибка;
- правило;
- возможная коррекция;
- объект внимания.

Рабочая формула:

```text
PNA -> временная сборка -> внутренний прогон -> изменение поведения
```

---

## 7. Связь с ΔT

ΔT — окно выбора.

Сон может расширять обучение за пределы текущего ΔT.

Воображение создаёт симулятивное ΔT.

Рефлексия удерживает ΔT на уровне правил.

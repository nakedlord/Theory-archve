## Назначение

Этот файл переносит старую связку воображения, внутреннего прогона и R2/R3.

Mental simulation — это внутреннее проигрывание возможных действий, траекторий и последствий.

---

## 1. Главная идея

Mental simulation позволяет учиться без немедленного действия в реальности.

Это снижает цену ошибки.

Формула:

```text
память -> возможная траектория -> ожидаемый результат -> коррекция выбора
```

---

## 2. Связь с R2

R2 удерживает последовательность действий.

Mental simulation позволяет проиграть эту последовательность до действия.

Примерная схема:

```text
если я сделаю A -> потом B -> возможен результат C
```

---

## 3. Связь с R3

R3 анализирует не только траекторию, но и правило, которое породило траекторию.

Mental simulation может поддерживать R3, если система проигрывает не только действие, но и альтернативное правило.

Формула:

```text
старое правило -> воображаемый результат -> ошибка -> новое правило
```

---

## 4. Связь с PNA

PNA может быть временным окном, где удерживается симулируемая ситуация.

Внутри PNA могут удерживаться:

- стартовая ситуация;
- возможное действие;
- ожидаемая реакция среды;
- альтернативная траектория;
- критерий результата.

---

## 5. Связь с ΔT

Mental simulation может расширять ΔT.

Вместо выбора только между текущими действиями система выбирает между возможными траекториями.

---

## 6. Риски

Нельзя считать всякое воображение R3.

Воображение может быть:

- хаотическим;
- фантазийным;
- эмоциональным;
- защитным;
- планирующим;
- рефлексивным.

Только последняя часть связана с перепрошивкой правила.

---

- mental simulation;

- planning;
- counterfactual thinking;
- model-based decision making;
- imagination and working memory.

# Working Memory and Child Development — PNA / R Theory Migration Draft

Статус: migration draft / needs source verification

## Назначение

Этот файл переносит старую связку PNA, рабочей памяти, детского развития и переходов R1/R2/R3.

Файл не является доказанным нейрокогнитивным каноном.

---

## 1. Главная идея

Развитие ребёнка можно читать как постепенное расширение способности удерживать:

- действие;
- результат;
- последовательность;
- причину;
- ошибку;
- правило;
- альтернативу.

Эта способность связана с рабочей памятью, вниманием, речью и внешним скаффолдингом.

---

## 2. R1 в детском развитии

На R1-уровне ребёнок удерживает простую связку:

```text
действие -> ближайший результат
```

Обучение часто идёт через повторение, реакцию взрослого и непосредственный результат.

---

## 3. R2 в детском развитии

На R2-уровне ребёнок начинает удерживать последовательность:

```text
сначала -> потом -> результат
```

Для этого нужны:

- рабочая память;
- внимание;
- повторяемые сценарии;
- поддержка взрослого;
- речь;
- предсказуемая среда.

---

## 4. R3 в детском развитии

На R3-уровне ребёнок начинает удерживать ошибку и правило:

```text
я сделал так -> получилось не так -> почему -> как изменить правило
```

Это требует более сложной работы внимания, речи и памяти.

---

## 5. Скаффолдинг

Взрослый может временно выполнять часть R2/R3-функции за ребёнка.

Он удерживает:

- цель;
- последовательность;
- правило;
- ошибку;
- причину;
- следующий шаг.

Постепенно ребёнок интериоризирует часть этой структуры.

---

## 6. Связь с PNA

PNA может быть окном, в котором ребёнок удерживает текущую ситуацию.

По мере развития рабочей памяти и внимания PNA может становиться более устойчивым и связным.

Осторожная формула:

```text
working memory development -> stronger temporary situation window -> possible support for R2/R3
```

---

## 7. Связь с речью

Речь помогает удерживать структуру за пределами одного мгновенного окна.

Через речь ребёнок может повторить правило, назвать ошибку и удержать последовательность.

---

## 8. Что нужно проверить

- развитие рабочей памяти у детей;
- развитие причинного объяснения;
- private speech;
- роль взрослого;
- executive function;
- attention control;
- narrative development.

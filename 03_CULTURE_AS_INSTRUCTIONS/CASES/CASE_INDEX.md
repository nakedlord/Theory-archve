# Culture as Externalized Instructions Case Index

Статус: active case index
Дата: 2026-04-25

## Назначение

Этот файл индексирует кейсы Culture as Externalized Instructions.

Кейс не является доказательством теории. Кейс показывает, как рамка культуры как сети инструкций работает на материале и какие проверки нужны дальше.

---

## 1. `historical_person_as_instruction_node.md`

### Статус

```text
draft case / supports mythologization v1.1
```

### Суть

Историческая личность после смерти может стать культурным узлом передачи.

Передаётся не полный человек, а набор признаков, различений, образов и правил, которые лучше копируются.

### Что усиливает

- мифологизация как изменение весов передачи;
- личность как узел сети;
- упрощение агентности в передаваемый образ.

### Что не доказывает

- конкретный исторический кейс;
- универсальность механизма для всех исторических личностей;
- точную структуру передачи без источников.

## 4. `mark_gospel_instruction_read.md`

### Статус

```text
migration draft / needs close textual verification
```

### Суть

Евангелие от Марка читается как ранний нарративный слой, где образ Иисуса передаётся через сцены, конфликты, непонимание, страдание и развязку.

### Что усиливает

- текст как instruction network;
- нарратив как носитель R2/R3-инструкций;
- возможность анализа конкретного евангельского корпуса.

### Что не доказывает

- приоритет Марка как исторической истины;
- конкретные эпизоды без текстового анализа;
- историческую реконструкцию.

## 6. `contradictions_and_instruction_layers.md`

### Статус

```text
migration draft / methodological case
```

### Суть

Нестыковки в тексте могут указывать на разные слои: исторический, нарративный, инструктивный, мифологический, институциональный.

### Что усиливает

- историческое ядро;
- текстовую стратификацию без источников.

### Что усиливает

- отделение инструкции от события;


## 1. Правило проверки

Для каждого утверждения нужно фиксировать:

```text
claim -> source -> what source says -> instruction-network reading -> confidence status
```

Источник не доказывает теорию напрямую.

Источник может поддерживать отдельный мост.

---

## 2. Confidence statuses

### supported

Источник прямо поддерживает factual claim.

### plausible

Источник совместим с instruction-network reading, но не доказывает его.

### contested

Есть значимые альтернативные объяснения.

### speculative

Идея перенесена из наших рассуждений и требует источников.

### needs source

Нужен источник до использования в сильном выводе.

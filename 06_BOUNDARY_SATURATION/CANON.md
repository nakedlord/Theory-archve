# Boundary Saturation — Black Hole Model

Статус: speculative formal model.

## Короткое определение

Чёрная дыра рассматривается как режим насыщения границы: горизонт возникает, когда внутренняя генерация и актуализация корреляций превышает пропускную способность границы области.

## Базовая схема

Пусть есть область B радиуса R.

Граница области имеет пропускную способность:

```text
C_{∂B} = Σ c_e
```

где `c_e` — пропускная способность ребра cut.

При росте плотности внутренних корреляций пропускная способность границы деградирует:

```text
c_e = c0 · g(ρ_Ze), где g′ < 0
```

## Критерий горизонта

Горизонт возникает, когда:

```text
τ_scr << τ_escape
и
I_dot_B ≳ C_{∂B}
```

То есть внутренняя скорость скремблинга и производства корреляций становится выше, чем способность границы выпускать информацию наружу.

## Интерпретация

Обычный объект имеет малую плотность Z-связей и видим как объём.

Чёрная дыра — режим, где внешне доступная энтропия насыщается по границе:

```text
S_max ~ A / l^2
```

## Статус

Модель является спекулятивной и требует формализации: порог E/R, аналог r_s, связь с Page-like поведением и проверяемыми следами.

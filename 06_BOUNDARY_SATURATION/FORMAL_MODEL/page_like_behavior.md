# Page-like Behavior — Boundary Saturation Migration Draft

Статус: speculative migration draft / needs formalization

## Назначение

Этот файл переносит старую линию Boundary Saturation про Page-like поведение корреляций.

Файл не является физическим предсказанием в строгом смысле.

Он фиксирует candidate-направление для будущей формализации.

---

## 1. Исходная идея

Если чёрная дыра читается как режим насыщения границы, то внешне доступные корреляции должны иметь характерную динамику.

Интуиция:

```text
внутренние корреляции -> граничный bottleneck -> внешне доступные корреляции -> Page-like dynamics candidate
```

---

## 2. Почему Page-like мотив возникает

В стандартной физической дискуссии Page curve связана с вопросом, как информация распределяется в процессе испарения чёрной дыры.

В нашей модели этот мотив появляется осторожно: если граница является главным интерфейсом доступности, то динамика выхода корреляций может быть не линейной и не просто локальной.

---

## 3. Boundary Saturation reading

В терминах Boundary Saturation:

```text
boundary load changes -> accessible correlation structure changes
```

То есть при изменении состояния границы меняется не только количество выходящего, но и структура доступных корреляций.

---

## 4. Что это может означать

Candidate-идея:

- ранняя внешняя доступность корреляций может быть низкой;
- затем структура доступных корреляций меняется;
- позднее внешняя система может получать больше информации о внутренней структуре;
- это может напоминать Page-like behavior.

---

## 5. Что это не доказывает

Этот файл не доказывает:

- Page curve;
- сохранение информации;
- механизм Хокинговского излучения;
- unitary evaporation;
- новую физику чёрных дыр.

---

## 6. Что нужно для формализации

Нужно определить:

1. что считается корреляцией;
2. где она хранится;
3. как она становится внешне доступной;
4. как меняется boundary load;
5. как это связано с compactness;
6. есть ли наблюдаемо отличимое следствие.

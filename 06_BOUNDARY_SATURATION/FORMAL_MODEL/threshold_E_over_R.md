# Formal Model Draft: threshold_E_over_R

Дата: 2026-04-25
Теория: Boundary Saturation
Статус: speculative draft

## Важно

Этот файл не является доказанной физической моделью.

Он фиксирует рабочий черновик для идеи:

```text
чёрная дыра как режим насыщения границы
```

Задача файла — отделить физическое утверждение от аналогии и подготовить модель к проверке.

---

## 1. Исходная интуиция

Есть область B радиуса R.

У области есть граница ∂B.

Если внутренняя генерация или актуализация корреляций становится выше пропускной способности границы, возникает режим насыщения.

---

## 2. Переменные

```text
B        область
R        радиус области
E        энергия внутри области
A        площадь границы
C_b      пропускная способность границы
I_dot    скорость внутренней актуализации корреляций
```

---

## 3. Простая запись границы

Если граница состоит из каналов выхода, то:

```text
C_b = sum(c_e)
```

В первом приближении число каналов связано с площадью:

```text
N_cut ~ A / l^2
```

Тогда грубо:

```text
C_b ~ c0 * A / l^2
```

---

## 4. Условие насыщения

Рабочая запись:

```text
I_dot >= C_b
```

Смысл: внутри области создаётся или актуализируется больше корреляций, чем граница может сделать доступными наружу.

---

## 5. Связь с E/R

Для чёрной дыры известный компактностный параметр можно записать как:

```text
chi = 2GE / (c^4 R)
```

Горизонтальный режим соответствует:

```text
chi >= 1
```

То есть порог можно выразить как:

```text
E / R >= c^4 / 2G
```

Boundary Saturation не должен заменять этот критерий.

Рабочая задача модели — переописать его как режим насыщения границы.

---

## 6. Минимальная безопасная формулировка

```text
boundary saturation candidate occurs when dimensionless compactness approaches the horizon threshold
```

Иначе:

```text
chi -> 1
```

можно читать как переход, где внешне доступное описание начинает определяться не объёмом, а границей.

## 7. Что модель пока не доказывает

Модель пока не доказывает:

- новую физику чёрных дыр;
- механизм испарения;
- внутреннее состояние вещества;
- замену общей теории относительности;
- точную микрофизику корреляций.
